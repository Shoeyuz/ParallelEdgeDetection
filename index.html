Project Outline:
My project will involve the comparison of classical edge detection algorithms. I will implement a serial version of the canny and sobel edge detection, after which I will reimplement them using parallelism. From here, we will carry out comparisons of the two. We will compare on multiple parameters: performance, performance benefit from transitioning over to parallelism, and quality of edge detection. Given enough time and succesful implementations I will attempt to implement the other two mainstream edge detection algorithms as well. The startup papers involve the mathematical foundations for computational edge detection which we will use for our implementation of the serial, while the parallelism we will try to implement through our own approach. We will compare our approach to the parallelism to the other existing papers on parallel edge detection with the existing algorithms to both verify their information and compare results across edge detection algorithms
